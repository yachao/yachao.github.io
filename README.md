# 隐私政策

**生效日期**: 2026 年 4 月 15 日

Whip App（以下简称"本应用"）重视用户隐私。本隐私政策说明我们如何收集、使用和保护您的个人信息。

## 1. 信息收集

### 1.1 我们不收集的信息
- 本应用**不会**收集您的个人身份信息
- 本应用**不会**收集您的位置信息
- 本应用**不会**与第三方共享任何数据

### 1.2 我们收集的信息

**运动数据**（仅用于功能实现）：
- 加速度计数据：用于检测甩动动作
- 陀螺仪数据：用于精确判断甩动强度
- 数据仅在使用时实时读取，**不会存储或上传**

**本地存储数据**：
- 使用次数统计：存储在设备本地，用于免费版每日 3 次限制
- 设置偏好：灵敏度、音量等设置
- 购买状态：通过 Apple StoreKit 验证

## 2. 信息使用

收集的信息仅用于：
- 实现甩动检测功能
- 播放对应的音效
- 提供触觉反馈
- 管理免费/专业版权限

## 3. 信息共享

**我们不出售、不交易、不共享您的任何个人信息给第三方。**

以下情况除外：
- Apple StoreKit 内购数据（由 Apple 处理）
- 法律要求的情况

## 4. 数据安全

所有数据存储在您的设备本地：
- 使用 iOS 系统安全机制保护
- 不上传到任何服务器
- 不使用第三方分析服务

## 5. 儿童隐私

本应用适合所有年龄段，包括 13 岁以下儿童：
- 不收集儿童个人信息
- 不包含不适内容
- 不需要家长同意

## 6. 第三方服务

本应用使用以下第三方服务：
- **Apple StoreKit**: 处理内购，遵循 Apple 隐私政策
- **Core Motion**: Apple 官方框架，数据不离开设备

## 7. 隐私政策更新

我们可能会更新本隐私政策：
- 更新后会在新版本中说明
- 继续使用视为接受新政策
